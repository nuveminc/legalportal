Termstore Configuration Settings

You will need to configure your Site Collection Term Store before you complete this section.
To do this you will need to create a new Term Store along with the following Term Sets - these are the base Taxonomy termsets that have been established.
After you have created the following Term Sets, you can create any number of DocTypes, Groups, or Issues.

Default Term Sets

  • DocType (no space in name)
  • Groups
  • Issues
To obtain the GUID, navigate to the Term Store or Term Set page. The Unique Identifier (GUID) is located at the bottom of the page.
Your Term Store Collection Id can be obtained from the Twitter Widgets page (login to access)
Your Term Set DocType GUID can be obtained from the Twitter Widgets page (login to access)
Your Term Set Groups GUID can be obtained from the Twitter Widgets page (login to access)
Your Term Set Issues GUID can be obtained from the Twitter Widgets page (login to access)
